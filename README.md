# 睡眠周期计算器

基于90分钟睡眠周期+15分钟睡前缓冲科学计算

## 功能介绍

1. 计算就寝时间：输入您希望醒来的时间，计算最合适的就寝时间。

2. 计算醒来时间：输入您计划上床的时间，计算最合适的醒来时间。

## 技术参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 睡眠周期 | 90 分钟 | 科学推荐的睡眠周期长度 |
| 入睡缓冲 | 15 分钟 | 入睡前所需的准备时间，计算就寝时间时提前扣除 |
| 充足睡眠周期数 | 5-6 个 | 最佳睡眠周期范围 |
| 休息不足提醒 | 1-2 个 | 睡眠时长较短，短暂休息 |

## 睡眠周期说明

- **1个周期**：1.5小时睡眠（短暂休息）
- **2个周期**：3小时睡眠（短暂休息）
- **3个周期**：4.5小时睡眠（快速恢复）
- **4个周期**：6小时睡眠（标准睡眠）
- **5个周期**：7.5小时睡眠（充足睡眠）
- **6个周期**：9小时睡眠（充足睡眠）

## 使用方法

### 场景一：我想在几点醒来？
1. 输入您希望醒来的时间
2. 点击"计算就寝时间"按钮
3. 系统会显示1-6个周期的建议就寝时间，并已扣除15分钟睡前缓冲

### 场景二：我想几点上床？
1. 输入您计划上床的时间
2. 点击"计算醒来时间"按钮
3. 系统会显示1-6个周期的建议醒来时间，并已加上15分钟入睡缓冲
